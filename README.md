# CVP Decision Intelligence System V4.0

這是一個功能強大的「本量利投」一體化決策支援系統，整合了 CVP 分析、蒙地卡羅風險模擬、投資 ROI 分析與 AI 決策報表。

## 核心功能
- **19 個決策模組**：涵蓋損益、模擬、投資、趨勢、風險等多個面向。
- **動態情境模擬**：即時測試 4 大變數對利潤的影響。
- **風險分析**：支持 10,000 次蒙地卡羅模擬及 VaR 計算。
- **現代化介面**：支援亮色/暗色切換，使用米白色面板設計。
- **零後端部署**：純 Vanilla JS 實作，可直接託管於 Vercel 或 GitHub Pages。

## 如何使用
1. 下載本專案中的 `CVP_Sample_Data_V4.xlsx`。
2. 填入您的業務數據。
3. 在頁面上方點擊「匯入 Excel」即可開始分析。

## 技術堆疊
- Tailwind CSS
- Chart.js v4
- SheetJS (XLSX)
- Vanilla JS (ES6+)
